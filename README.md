# currencies-list

## Installation and Setup Instructions:

Clone down this repository. 
`git clone git@github.com:malwina-mika/currencies-list.git`
You will need `node` and `npm` installed globally on your machine.  

### Installation:

`npm install`   

### To Start Server:

`npm start`  

### To Run Tests:  

`npm test` 

### To Visit App:

`http://localhost:8080/`
